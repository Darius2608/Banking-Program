# Banking-Program
I created a banking program to manage basic banking transactions acting like a ATM. which displays 4 options: show balance, deposit, withdraw and exit. User has to enter a number which corresponds to one of the options. Through this program, the user can check their bank balance, update their funds by depositing and withdrawing money and program loops till user decides to exit the program.
I used a while loop to keep the menu running and a switch case to handle the user's input for each option. I also added some if/else statement so that it won't let you withdraw more money than you actually have, and it prevents you from depositing negative amounts. Additionally, I used printf to make sure the bank balance always looks correct with two decimal places. Static methods was something I have learnt to keep the code organized by putting the deposit and withdraw logic into their own separate blocks of code to look neater and cause less confusion.
